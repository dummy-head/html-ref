Show status of a Redis service


## Usage
~~~
flyctl redis status <id> [flags]
~~~

## Options

~~~
  -h, --help   help for status
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~

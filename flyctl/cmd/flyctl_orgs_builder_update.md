Update an organization's remote builder image

## Usage
~~~
flyctl orgs builder update <org-name> <image_ref> [flags]
~~~

## Options

~~~
  -h, --help   help for update
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~

# _flyctl orgs create_

Create an organization

### About

Create a new organization. Other users can be invited to join the 
organization later.

### Usage
```
flyctl orgs create <org> [flags]
```

### Options

```
  -h, --help   help for create
```

### Global Options

```
  -j, --json                  json output

```

Reset WireGuard peer connection for an organization

## Usage
~~~
flyctl wireguard reset [org] [flags]
~~~

## Options

~~~
  -h, --help   help for reset
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~

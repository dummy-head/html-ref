The `restart` command will restart all running VMs. 

## Usage

~~~
flyctl restart [APPNAME] [flags]
~~~

## Options

~~~
  -h, --help   help for restart
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
